# Диагностика ошибки 404 на Vercel

## Нужные данные для диагностики:

1. **URL, на котором возникает ошибка:**
   - Например: `https://www.cristobal.pro/` или `https://ristobel.vercel.app/`

2. **Логи деплоя из Vercel:**
   - Зайдите в Vercel Dashboard → Ваш проект → Deployments → выберите последний деплой
   - Скопируйте логи сборки (Build Logs) и ошибки (если есть)

3. **Настройки проекта в Vercel:**
   - Framework Preset: должен быть "Next.js"
   - Root Directory: должен быть "." (корень проекта)
   - Build Command: должен быть "npm run build" или пусто (по умолчанию)
   - Output Directory: должен быть пустым или ".next"

4. **Информация о домене:**
   - Какой домен используется: `www.cristobal.pro` или `ristobel.vercel.app`?
   - Правильно ли настроен DNS для домена?

5. **Скриншоты:**
   - Скриншот страницы с ошибкой 404
   - Скриншот настроек проекта в Vercel (Settings → General)
   - Скриншот логов деплоя (если есть ошибки)

## Альтернативное решение:

Попробуйте также добавить в `vercel.json` rewrites для обработки корневого пути.
