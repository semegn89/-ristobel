## 1. ЦВЕТОВАЯ ПАЛИТРА И ТИПОГРАФИКА

### 1.1 Расширенная цветовая палитра

**Текущая палитра:**
- Primary: #1e3a5f (темно-синий)
- Secondary: #4a5568 (серый)

**Предлагаемые улучшения:**

#### Основные цвета:
```css
Primary:
- dark: #0f1e3a (более глубокий, премиальный синий)
- DEFAULT: #1e40af (яркий синий - акцент)
- light: #3b82f6 (светлый синий)
- accent: #06b6d4 (циан для акцентов)

Secondary:
- dark: #1f2937 (угольный)
- DEFAULT: #6b7280 (нейтральный серый)
- light: #e5e7eb (светло-серый)

Accent colors:
- success: #10b981 (зеленый для успеха)
- warning: #f59e0b (янтарный)
- error: #ef4444 (красный)
```

#### Градиенты:
- Hero: `linear-gradient(135deg, #0f1e3a 0%, #1e40af 50%, #3b82f6 100%)`
- Cards: `linear-gradient(135deg, #ffffff 0%, #f8fafc 100%)`
- Buttons: `linear-gradient(135deg, #1e40af 0%, #3b82f6 100%)`

### 1.2 Типографика

**Текущее:** Стандартные шрифты системы

**Предлагаемое:**
```css
Headings: 'Inter' или 'Poppins' (Google Fonts)
- H1: 3.5rem (56px), font-weight: 700, line-height: 1.2
- H2: 2.5rem (40px), font-weight: 600, line-height: 1.3
- H3: 1.875rem (30px), font-weight: 600, line-height: 1.4

Body: 'Inter' или 'Roboto'
- Base: 1rem (16px), font-weight: 400, line-height: 1.6
- Large: 1.125rem (18px), line-height: 1.7
- Small: 0.875rem (14px), line-height: 1.5

Letter spacing:
- Headings: -0.02em (более компактно)
- Body: 0 (стандарт)
```

## 3. КАРТОЧКИ И КОМПОНЕНТЫ

### 3.1 Карточки услуг

**Текущее:**
- Простые белые карточки
- Тень при hover
- Базовые анимации

**Предлагаемое:**

#### Дизайн:

**Предлагаемое:**
